zoonyoungc ♥ 4 October 2019 Unity 2 min

ARCore Augmented Image (Android)


Unity로 Augmented Image 구현 (2019년 10월 기준)


Sample Scene 빌드 하기

  • Part-1 에서 한 것처럼 새 프로젝트 만들기.
  • 아래 경로에 있는 이미지 기반 증강현실 샘플 Scene을 더블 클릭.
    Assets/GoogleARCore/Examples/AugmentedImage/Scenes/
  • 기기 연결하고 Build and Run, 빌드가 끝나면 기기를 움직여 참조 이미지 를 찾기
  • 샘플 Scene의 참조 이미지는 아래 경로상에 있음.
    Assets/GoogleARCore/Examples/AugmentedImage/Images


Sample Scene 빌드로 본 타겟(참조) 이미지와 증강현실 표현

  1. ARCore에서 제공한 샘플 이미지

  2. 빌드 해서 확인한 증강현실 표현

    ※ 모니터 상에 이미지를 띄우고 빌드한 앱을 실행하여 카메라로 비춤
    ※ 4곳의 모서리에 액자 틀이 생기는 이미지 기반의 증강현실


준비사항

    이미지 기반의 증강현실이니까 참조 이미지가 필요.
    우선 빌드해 본 Augmented Image Sample Scene를 기반으로 작업.


타겟이 될(참조가 될) 이미지에 대한 요구사항

  • 카메라의 프레임 25% 이상 채울 것.
  • 평평해야 할 것.
  • 부분적으로 가리거나, 각도가 비스듬하거나, 카메라를 너무 빠르게 움직이지 말것.


참조할 이미지의 좋은 사례

  • PNG, JPEG를 쓰고 JPEG는 과도한 압축은 하지 말 것.
  • 컬러, 흑백 상관없음.
  • 해상도는 300 * 300 픽셀 이상.
  • 굳이 고해상도일 필요 없음.
  • 이미지가 뚜렷하지 않고 반복적인 패턴 같은 것은 피할 것.


참조 이미지로 Database 만들기

  • 내가 원하는 이미지 상에 증강현실을 표현하기 위해 내 이미지들로 데이터베이스 만들기.
  • Project 창에서 내가 사용할 참조 이미지들을 선택하고 우클릭하여
    Create -> Google -> ARCore -> AugmentedImageDatabase
  • 이미지를 모아둔 같은 폴더 안에 새 이미지 데이터베이스가 생성됨.
  • 새 데이터베이스 Inspector 창으로 보면 각각의 이미지들의 평가해서
    점수를 주는데 75 점 이상 받아야 함.


데이터베이스 사용하기

  • 아래 경로에 있는 AugmentedImagesSessionConfig 열기.
    Assets/GoogleARCore/Examples/AugmentedImage/Configurations
  • Inspector 창을 보면 Augmented Image Database 있는데 여기에 내가 만든 데이터베이스 넣어주기.


Code

  • 아래 경로에 있는 AugmentedImageExampleController.cs 를 보면
    Assets/GoogleARCore/Examples/AugmentedImage/Scripts

  • Update 스크립트 내부에서 아래 함수가 돌아가며 이미지를 찾아냄(Tracking).

Session.GetTrackables<AugementedImage>(m_TempAugmentedImages, TrackableQueryFilter.Updated);

  • 이미지를 찾아내어 추적을 시작하면 Anchor 포인트가 생김.
  • Augmented Image Visualizer 에 Prefab을 연결하여 Anchor 에 걸어줌.
  • 그 Prefab이 위 사진에 나온 것처럼 액자 틀로 표현된 것.
  • Prefab을 수정하기 위해서는 AugmentedImageVisualizer.cs 와 3D 오브젝트 변경하면 됨.
