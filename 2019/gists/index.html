Hugo 사이트 안에 Gist 작성 (내장 shortcode 이용)


Gist 작성

  • Gist에서 언어에 따른 코드를 작성. (e.g. 파일명.cs)
  • Create public gist 클릭
  • Embed 쪽에 주소 생성된거 복사해도 되지만 필요한건 주소창에 다 나옴
  • 주소창을 보면 http://gist.github.com/<your ID>/<your-gist-address>/


.md 에서 바로 shortcode 작성

  • 기존에는 shortcodes 폴더에 .html을 작성하였으나 언제부턴가 Hugo 에서 shortcode 기본 제공
  • gist가 들어가야 할 위치에 shortcode 작성
  • { {< gist <your-ID> <your-gist-address> >} } 이런 형태로 작성


작성 완료

  • 아래 gist는 이런형태로 작성된 것 -> { {< gist zoonyoungc e0017f5c09f0d7bf2ecb33f33b730035 >} }



문제점

  • gist 코드의 일부만(정해진 라인만) 보게끔 하는 방법은?
