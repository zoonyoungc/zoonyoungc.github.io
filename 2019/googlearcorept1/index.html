Google ARCore Unity-Part 1

Written by zoonyoungc with ♥ on 4 October 2019 in Unity 2 min

Google by ARCore SDK for Unity (Android)


Unity로 간단히 ARCore 시작하기 (2019년 10월 기준)


요구사항

  • Hardware

    • ARCore 지원 기기 확인
    • 빌드 할 기기와 연결 상태 확인

  • Software (2017.4.26f1 이후 버전)

    • Android Build Support.
    • ARCore SDK for Unity 는 LWRP(URP)를 지원하지 않음.
    • Unity 2019 이상 사용중이면 추가 패키지 필요 (Multiplayter HLAPI, Legacy Input Helper).
    • ARCore SDK for Unity 1.12.0 or Later.
    • Android SDK 7.0 (API Level 24) or Later(Nougat 랑 그 이후 버전).


ARCore SDK for Unity

  • 구글이 제공하는 ARCore github에서 arcore-unity-sdk-1.12.0.unitypackage 다운로드.


새 프로젝트 만들고 SDK 임포트 하기

  • 새 프로젝트 만들 때 3D 선택.
  • Unity 2019는 Window -> Package Manager 로 들어가서 추가 패키지 설치
    • Multiplayer HLAPI (CloudAnchors 때문인 것 같고)
    • XR Legacy Input Helpers (Instant Preview 때문인 것 같고)
  • ARCore SDK for Unity 임포트
    • Asset -> Import Package -> Custom Package
    • 다운로드 받아둔 arcore-unity-sdk-1.12.0.unitypackage 선택.
    • 패키지 임포트 내용 주욱 뜨면 그대로 모두 임포트.


Sample Scene 불러오기

  • Unity의 Project 창에서 아래 경로 상 있는 HelloAR Scene 더블클릭.

    Assets/GoogleARCore/Examples/HelloAR/Scenes/


프로젝트 환경 설정

  • File -> Build Settings 에서 Android 로 플랫폼 변경.
  • Add Open Scenes 버튼 눌러서 현재 Scene을 빌드 지정.

  • Player Settings 에서 기본 세팅 변경.

    경로 입력 내용
    Player Settings > Other Settings > Rendering Auto Graphics API 체크해제
    Player Settings > Other Settings > Package Name 패키지 이름 변경 (e.g. com.example.helloAR)
    Player Settings > Other Settings > Minimum API Level 안드로이드 7.0 Nought(API Level 24) 이상
    Player Settings > XR Setting > ARCore Supported Enable

  • Auto Graphics API 체크해제 하면서 목록 상에 Vulkan 이 있으면 삭제(Vulkan 체크한 후 마이너스버튼 클릭)


Build and Run 하여 샘플 앱 만들기

  • 연결한 기기의 개발자 옵션의 디버깅 활성화.
  • Unity 상에 Build Settings -> Build and Run 클릭.
  • 생성할 APK 파일명 정하고 빌드 완료 되면 ARCore 샘플 앱을 실행.
  • 평평한 바닥을 찾게 되면 Anchors가 생성됨.
  • 그 위를 탭하면 Android의 심볼인 Andy가 생성됨(원래는 Andy였으나 바뀌었음).
