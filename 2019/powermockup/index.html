Power Mockup 파워목업 파워모크업

zoonyoungc ♥ 17 December 2019 PowerMockup 2 min

PowerMockup 시작하기


  • PowerMockup 파워목업
    • 웹을 기획하고 설계하는 과정에서 필요한 화면을 정의하기 위하여 사용하는 툴
    • 800개 이상의 웹 설계 관련 Wireframe과 Icon들의 UI Library
    • Powerpoint가 있고 다룰줄 알면 되는 간단한 Add-On


요구사항

  • Software
    • OS : Windows
    • 기존 MS Office의 PowerPoint의 Add-On 형태로 설치
    • Microsoft PowerPoint 2007, 2010, 2013 or 2016, Office 365 가능


  • 참고사항

    • Free Trial Version : 무료라서 사용할 수 있는게 턱없이 적음
    • Full Version : $59.99 ~ $2,099.50

  • 다운로드 페이지

    • https://www.powermockup.com/download - 페이지에서 Download Free Trial 클릭


첫화면

  • Add-On 설치 이후 자동으로 PowerPoint가 실행되며 아래와 같은 사용방법이 작성된 가이드 PPT 가 실행됨
  • 만약 실행되지 않으면 시작 -> PowerMockup -> Welcome to PowerMockup.ppt 실행하면 됨


사용방법

  • 처음 실행되는 가이드 PPT 를 잘 보면 사용방법이 다 나와있음 1. 상단 탭바에서 PowerMockup 선택하면 자동으로 오른쪽 창에 UI 들이 나열
    2. 오른쪽에 창이 생기지 않는다면 Show Shapes Panel 선택
    3. UI들을 가운데 창으로 Drag&Drop으로 사용


결론

  • 장점 - 사용하기 편하다. 쉽다.
  • 단점 - Add-On의 템플릿 종류가 많아서인지 화면이 느려지는게 느껴지며 무료버전은 쓸게 없다! (죄다 비활성)


개인 라이센스(Free Individual License)

  • Free PowerMockup License
    • https://www.powermockup.com/order/free-license
  • 위 페이지 내용에 따르면 블로거, 저널리스트, MS MVPs 를 대상으로 개인 라이센스$(59.99)를 무료로 제공해줌.
  • 특히 블로거인 경우의 조건은
    • 정기적인 게시물과 합리적인 방문자 수
    • 블로그 내용이 웹, 모바일 혹은 데스크탑 어플리케이션 개발 관련
    • 도용한 내용이 아니어야 하며
    • 쉽게 찾을 수 있어야 함
  • 페이지에 나온 이메일로 풀 네임과 포스트 링크를 작성해서 보내면 라이센스 키를 보내준다고 함
  • 이 페이지를 링크로 보내려고 하는데 아마 안될 것 같…
