Google ARCore Unity-Part 4

zoonyoungc ♥ 13 October 2019 Unity 1 min

ARCore Augmented Image Sample App Code (Android)


ARCore의 Augmented Image Code 관련 문서는 따로 없기 때문에 Part 3 기준으로 내용 작성.


Augmented Image 구성

  • Assets/GoogleARCore/Examples/AugmentedImage/Scene 안에 Scene을 실행
  • 하이어라키 구성.
  • HelloAR 과 다른 점은 아래와 같음.
게임 오브젝트
설 명
ExampleController AugmentedImageExampleController.cs 스크립트를 포함하고 있는 게임오브젝트. AR Scene 전체를 관리함.
이미지위에 보여줄 Prefab을 관리하는 AugmentedImageVisualizer.cs 와 타겟 이미지를 찾기 전까지 계속 보여줄 가이드 UI 인 FitToScanOveray 가 있음.


Code

  • HelloAR 과는 다르게 Update() 항목에 _UpdateApplicationLifeCycle() 함수 내용을 직접 넣어놈. 업데이트 주기마다 화면슬립과 image tracking 을 확인함. image tracking이 되면 Anchor 를 생성하여 가상 오브젝트를 타겟 이미지위에 생성. tracking을 못하면 FitToScanOveray 가 가이드 UI 로써 계속 생성되어 있음.
  • AugmentedImageExampleController.cs 코드 중 일부


  • AugmentedImageVisualizer.cs 는 4개의 액자 틀을 보여주게 하는 스크립트. 이것을 수정하여 내가 원하는 3D 컨텐츠를 보여줄 수 있게 함. 액자틀에 들어갈 컨텐츠와 위치와 방향을 수정하면 됨.


  • 프로젝트가 변경되어 ARCore 포스트는 여기에서 종료
